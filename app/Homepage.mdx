### welcome.mdx
```markdown
{{ <span className="emphasis">Hey there **{'{'+'{ props.who }'+'}'}** ‚úåüèº</span> }}
```

### app.jsx
```jsx
{{ <span className="emphasis">import Welcome from './welcome.mdx';</span> }}
 
ReactDOM.render(
{{ <span className="emphasis">{`  <Welcome who="${props.who || 'Monkey Magic'}" />,`}</span>}}
{{`  document.getElementById('app')`}}
);
```

### Rendered:
```html
<div id="app">
{{ <span className="emphasis">{`  <p>Hey there <strong>${props.who || 'Monkey Magic'}</strong> ‚úåüèº</p>`}</span>}}
</div>
```

## What else can it do?

Use JSX within Markdown. Plus import objects such as shared snippets, fancy library functions or even whole React components, using YAML front-matter:

```markdown
---
{{<span className="emphasis">{'imports:'}</span>}}
{{<span className="emphasis">{'  numberFormat: number-format'}</span>}}
{{<span className="emphasis">{'  Button: ui/button'}</span>}}
---

You are visitor number {{<span className="emphasis">{'{{ numberFormat(1000000) }}'}</span>}}!

{{<span className="emphasis">{'<Button path="/browser-info">Details</Button>'}</span>}}
```

Destructured imports, with some added single quotes:

```markdown
---
imports:
{{<span className="emphasis">{'  \'{ window, document }\': global'}</span>}}
---

Hello, your user agent is {{<span className="emphasis">{'{{ window.navigator.userAgent }}'}</span>}}.
```
